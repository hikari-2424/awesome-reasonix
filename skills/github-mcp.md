# github

GitHub API MCP 服务器。通过 MCP 协议在 Reasonix 中直接操作 GitHub 仓库。

## 安装

```bash
reasonix mcp add github -- npx -y @modelcontextprotocol/server-github
```

## 功能

- 仓库管理（创建、Fork、Clone）
- Pull Request 和 Issue 操作
- 代码搜索
- 文件读写
