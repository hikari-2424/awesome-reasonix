# filesystem

安全的文件系统访问 MCP 服务器。通过 MCP 协议让 Reasonix 读取、写入、搜索本地文件。

## 安装

```bash
reasonix mcp add filesystem -- npx -y @modelcontextprotocol/server-filesystem /path/to/dir
```

## 功能

- 读取和写入文件
- 目录列表和搜索
- 可配置允许访问的目录范围
