# text-polish

Process text according to the operation type.

## Operations

### polish
- Fix grammar and typos
- Improve fluency
- Preserve original meaning
- Flag major changes

### translate
- Auto-detect Chinese ↔ English
- Preserve technical terms (API, SDK)
- Maintain formatting (markdown / plain text)

### summarize
- Output a 3-sentence version + 1-sentence version
- Preserve key numbers and names

### rewrite
Target styles:
- `professional` — formal business
- `casual` — relaxed conversational
- `concise` — ultra-short
- `tech-doc` — technical documentation

## Rules
- Don't add facts not in the original
- Note style changes when rewriting
- Preserve code blocks as-is
