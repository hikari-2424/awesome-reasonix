# data-convert

Convert between common data formats.

## Supported Formats
JSON, YAML, CSV, TOML, XML, Markdown tables

## Operations

### convert
Read source file, detect format, convert to target format.

### batch
Scan directory for matching files, convert all, keep original names.

### preview
Show first 20 lines of conversion, flag potential data loss.
