# readme-gen

Analyze project and generate professional README.md.

## Operations

### generate
1. Scan project structure, dependencies, entry points
2. Extract API/exports from code
3. Generate README with install, usage, API, config sections

### update
1. Read existing README
2. Compare with project changes
3. Update only changed sections, preserve hand-written content
