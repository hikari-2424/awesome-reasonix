# disk-cleaner

扫描指定目录，找到可清理的冗余文件。

## 操作

### scan（扫描）
1. 运行 `dir /s` 或 `ls -laR` 扫描目标目录
2. 分类标记：
   - `.tmp` `.temp` `.cache` `.log` → 临时文件
   - `node_modules` `__pycache__` `.pytest_cache` → 依赖缓存
   - `*.bak` `*.old` `*~` → 备份文件
   - 文件名+大小相同的 → 重复文件
   - 超过 100MB 的 → 大文件
3. 输出分类统计表：类型、数量、总大小

### clean（清理）
1. 基于 scan 结果展示清理计划
2. 用户确认后逐类删除
3. 先删最安全的（临时文件），后删需确认的（重复文件）

### duplicate（查重）
1. 对比文件名 + 文件大小
2. 列出疑似重复，标注保留建议（保留最近修改的）
3. 不自动删除

## 规则
- **所有删除操作先展示计划，经用户确认后执行**
- 不操作系统目录（C:\Windows、/System、/Library）
- 不删除 `.git/` 内的文件
- 大文件只报告不自动删除
- 跳过正在使用的文件（被锁定的文件）
