# text-polish

Обработка текста в соответствии с типом операции.

## Операции

### polish (редактирование)
- Исправление грамматики и опечаток
- Улучшение беглости
- Сохранение исходного смысла
- Отметка значительных изменений

### translate (перевод)
- Автоопределение китайского ↔ английского
- Сохранение технических терминов (API, SDK)
- Сохранение форматирования (markdown / обычный текст)

### summarize (реферирование)
- Версия из 3 предложений + версия из 1 предложения
- Сохранение ключевых цифр и имён

### rewrite (переписывание)
Целевые стили:
- `professional` — деловой
- `casual` — разговорный
- `concise` — сверхкраткий
- `tech-doc` — техническая документация

## Правила
- Не добавлять факты, отсутствующие в оригинале
- Отмечать смену стиля при переписывании
- Сохранять блоки кода без изменений
