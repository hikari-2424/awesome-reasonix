# data-convert

在常见数据格式之间转换，保持数据完整性。

## 支持格式

| 格式 | 扩展名 |
|------|--------|
| JSON | .json |
| YAML | .yaml, .yml |
| CSV | .csv |
| TOML | .toml |
| XML | .xml |
| Markdown 表格 | .md（表格部分） |

## 操作

### convert（转换）
1. 读取源文件，检测格式
2. 解析为中间结构
3. 转换为目标格式并输出

### batch（批量转换）
1. 扫描目录中所有匹配格式的文件
2. 逐一转换，保持原文件名、改扩展名
3. 输出转换报告：成功/失败/跳过的数量

### preview（预览转换）
1. 只展示转换后的前 20 行
2. 标注数据类型推断（字符串/数字/布尔/null）
3. 警告可能的数据丢失（如 CSV 不支持嵌套）

## 规则
- 不修改原文件（输出新文件或打印到终端）
- 嵌套结构转 CSV 时展平，并警告用户
- CSV 自动推断分隔符（逗号/制表符/分号）
- XML 属性映射为 `@attrName` 键
- 大文件（>1MB）先确认再处理
